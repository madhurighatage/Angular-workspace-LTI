State of form elements:


pristine: This property returns true if the element's contents have not been changed.
dirty: This property returns true if the element's contents have been changed.
touched: This property returns true if the user has visited the element.
untouched: This property returns true if the user has not visited the element.
valid: This property returns true if each element has a valid data ( works for all form fields )

angular does not change form status to pristine, when the user changes UI . so don't rely on prestine manually- in component this.f.markAsPristine();
